# Fieldnote

Fieldnote turns messy voice memos into usable written documents. It is designed for builders who think out loud and want the output to become notes, code snippets, or action lists.

## Features

- Voice memo upload and transcription-ready structure.
- Clean note output with summary and next steps.
- Lightweight archive for past recordings.

## Lessons Learned

The product became more useful when the interface focused on the post-recording workflow instead of the recording itself.
