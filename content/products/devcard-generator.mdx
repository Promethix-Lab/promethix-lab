# DevCard Generator

DevCard Generator creates shareable SVG profile cards from repository activity. It is a small CLI utility built for developers who want a visual project summary.

## Features

- Repository activity parsing.
- SVG profile-card output.
- Configurable labels and stat sections.

## Lessons Learned

Small CLI tools need excellent defaults. If the first generated card looks useful, users are more likely to customize it later.
