# Pocket Ledger

Pocket Ledger is a one-screen budgeting app for people who quit the last five budgeting apps. It keeps the core workflow manual, fast, and visible, so users can understand where money moved without connecting bank accounts.

## Features

- Manual transaction entry tuned for speed.
- Live balance, income, spending, and category totals.
- Small dashboard summaries that update instantly.

## Lessons Learned

Finance tools do not need to be complicated to feel useful. The fastest win was reducing the number of decisions a user has to make before logging the next transaction.
