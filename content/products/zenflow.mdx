# Zenflow

Zenflow is a browser-based focus timer with a calm interface, basic session stats, and optional ambient loops. It is intentionally small and easy to restart.

## Features

- Focus timer with session state.
- Simple statistics log.
- Ambient mode designed for long work blocks.

## Lessons Learned

A timer becomes more useful when the reset and restart flow feels frictionless. The interface has to stay quiet when the user is already focused.
